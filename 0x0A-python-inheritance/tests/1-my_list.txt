=======================
How to Use 1-my_list.py
=======================

This class defines a list method ``print_sorted(self)``

Usage
=====

``print_sorted(self)`` prints out a sorted list of integers

::
    import module.
    >>> MyList = __import__('1-my_list').MyList

::
    create class instance.
    >>> my_list = MyList()

::    
    add elements to list.

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

::
    print list.
    >>> print(my_list)
    [1, 4, 2, 3, 5]

::
    apply method.

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

::
    remove from list.

    >>> my_list.remove(5)
    >>> my_list.remove(2)
    >>> my_list.print_sorted()
    [1, 3, 4]
